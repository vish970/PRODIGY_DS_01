## Task-01: Population Distribution Visualization

###  Objective
Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable, such as the distribution of ages or genders in a population.  
For this task, we used **World Bank Population Data (1960–2025)**.

### Dataset
- Source: [World Bank – Population, total](https://data.worldbank.org/indicator/SP.POP.TOTL)
- Indicator: SP.POP.TOTL (Total Population, midyear estimates)

###  Tools Used 
- Python (Pandas, Matplotlib, Seaborn)  

###  Visualizations
1. **Histogram of Population Distribution (2020)**  
   - Shows how countries are distributed across population ranges.  
   - Most countries fall below 50M population.

2. **Bar Chart of Top 10 Most Populous Countries (2020)**  
   - Highlights China, India, USA as global demographic leaders.

### Insights
- Population distribution is highly skewed.  
- A few countries account for a large share of the global population.  
- Histograms reveal frequency ranges, while bar charts highlight direct comparisons.

### Conclusion
This task demonstrates how to visualize population data using histograms and bar charts. It provides insights into demographic concentration and global distribution patterns.
